# Phase-Contract Logo Assets

This logo turns the repository's core mechanism into a compact symbol:

- Square brackets represent the execution boundary enforced by phase contracts.
- Three contract cards represent the fixed context window: `common`, `phase`, and `execution`.
- The gradient state line represents `planctl` moving one phase at a time through the contract chain.
- The final check node represents `complete` writing objective state back to `state.yaml` and `handoff.md`.

Files:

- `phase-contract-logo.svg` - horizontal product lockup with wordmark.
- `phase-contract-mark.svg` - standalone mark for avatars, favicons, and square placements.
